
极简待办 (Minimalist Task Flow) 📝
这是一个专为 iOS 平台设计的极简任务流转应用。本项目基于 Notion 看板逻辑开发，旨在通过最简单的“点击”交互，实现任务从“备忘”到“完成”的自动化管理。

🌟 设计思路
核心逻辑借鉴了高效的看板管理法（Kanban），但针对移动端进行了极致简化：

单向流转：摒弃繁琐的拖拽，通过简单的点击文字实现任务状态的平滑跳转。

自动化清理：引入“物理粉碎”机制，已完成的事项在 3 天后自动从库中删除，保持界面始终清爽。

爽快交互：针对 iOS 优化了 Q 弹的视觉反馈，模拟原生 App 的操作手感。

🛠️ 核心功能
三段式工作流：

备忘清单：记录灵感与琐事，点击即进入“进行中”。

进行中：聚焦当前任务，点击即标记为“已完成”。

已完成：记录战果，点击可撤回至“进行中”以修正误触。

极简输入：备忘页底部常驻输入框，回车自动添加并收起键盘，不中断思考流。

数据持久化：使用 localStorage 技术，确保任务数据安全存储在用户手机本地，无需联网账号。

PWA 体验：支持“添加到主屏幕”，全屏运行，无浏览器地址栏干扰。

🚀 使用方法
1. 部署到手机
在 iPhone 上使用 Safari 浏览器 访问本项目地址。

点击浏览器底部的 “分享” 图标。

在弹出的菜单中选择 “添加到主屏幕”。

在桌面点击生成的图标即可像原生 App 一样开启体验。

2. 交互指南
添加任务：在备忘页底部输入内容，按键盘“完成/确认”键。

推进状态：点击列表中的任何文字条目。

找回误触：在“已完成”页面点击该条目，它会重新回到“进行中”。

🏗️ 技术细节
Frontend: HTML5, CSS3, JavaScript (Vanilla JS)

Storage: LocalStorage API

iOS Optimization: PWA Meta Tags, Safe Area Inset Support, apple-touch-icon
